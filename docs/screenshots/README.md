# NutriMind AI - Screenshots Checklist Guide

This document lists the expected visual screenshots of the **NutriMind AI** platform. Developers can capture these screenshots of the running application and save them in this directory (`docs/screenshots/`) using the suggested filenames.

---

## Required Screenshots Checklist

### 1. Home / Landing Page
* **Filename:** `docs/screenshots/home.png`
* **Description:** The homepage showing the wellness platform intro, core feature cards (AI personalization, specialist slots, logs), and login/registration action buttons.

### 2. Login View
* **Filename:** `docs/screenshots/login.png`
* **Description:** The login modal or page styled with glassmorphism cards, prompting for email and password inputs.

### 3. Registration View
* **Filename:** `docs/screenshots/register.png`
* **Description:** The registration page presenting forms to fill name, email, password, and select user role (Client, Dietitian, Nutritionist).

### 4. Client Dashboard
* **Filename:** `docs/screenshots/dashboard.png`
* **Description:** The main dashboard view containing interactive Recharts graphs tracking daily water consumption, weight changes, calories tracker, and sleep durations.

### 5. Meal Planner
* **Filename:** `docs/screenshots/meal_planner.png`
* **Description:** The client page illustrating assigned meal programs, showing daily breakfast, lunch, dinner, and snack lists.

### 6. AI Recommendation
* **Filename:** `docs/screenshots/ai_recommendation.png`
* **Description:** The AI analysis panel illustrating computed daily calorie requirements and macronutrient partitions (carbs, protein, fat) generated by the Gemini engine.

### 7. Recipe Generator
* **Filename:** `docs/screenshots/recipe_generator.png`
* **Description:** The recipes builder interface with ingredient tags, cuisine selectors, and the output step-by-step instructions.

### 8. Chat Room
* **Filename:** `docs/screenshots/chat.png`
* **Description:** The messaging interface showcasing active chats between the client and dietitian or the Gemini AI nutritionist, complete with typing state indicators.

### 9. Dietitian Dashboard
* **Filename:** `docs/screenshots/dietitian_dashboard.png`
* **Description:** The specialist workspace listing assigned client profiles, their bio-details, and daily food logging charts.

### 10. Video Consultation
* **Filename:** `docs/screenshots/video_consultation.png`
* **Description:** The appointment scheduler page showing pending/approved slots, notes, durations, and virtual meeting links (Jitsi).

### 11. Admin Dashboard
* **Filename:** `docs/screenshots/admin_dashboard.png`
* **Description:** The administrator panel displaying system aggregates cards, user role update selectors, and profile delete buttons.
